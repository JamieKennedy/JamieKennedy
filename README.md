Full Stack web developer
------------------------
I'm a full stack web developer with a passion for building elegant and functional web applications. With several years of experience in the field, I've honed my skills in both front-end and back-end development, allowing me to create seamless user experiences from start to finish.

My expertise lies in HTML, CSS, JavaScript, and various web development frameworks, including React and NextJS. I'm also skilled in server-side technologies such as .Net Core and .Net Framework in both C# and VB.Net.

I'm constantly learning and experimenting with new technologies to stay up-to-date with the latest trends and advancements in the field. I believe in writing clean, testable, maintainable, and efficient code to ensure that my projects are both scalable and future-proof.

I'm always looking for new challenges and opportunities to collaborate with other developers, so feel free to get in touch!
